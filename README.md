# ☕ Anwar Coffee Shop — ETL Pipeline

A modular Python ETL project that simulates operations for **Anwar Coffee Shop** — a modern café serving coffee, tea, food, and merchandise.

This project generates realistic datasets, cleans and transforms them, and loads everything into a **MySQL database** for analytics and dashboards (e.g., Streamlit or Flask APIs).

---

# Installation

```bash
git clone https://github.com/yourusername/anwar-coffee-etl.git
cd anwar-coffee-etl
```

# Create a virtual environment

```bash
python -m venv .venv
source .venv/bin/activate
```

# Install dependencies

```bash
pip install -r requirements.txt
```
